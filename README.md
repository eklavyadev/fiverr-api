This is an unofficial api to fetch Fiverr seller's data

## How to run locally?

> **Prerequisites:**  
> python >= 3.9  
> pip >= 20.3

```zsh
git clone https://github.com/salmannotkhan/fiverr-api.git
cd fiverr-api
pip3 install -r requirements.txt
uvicorn main:app
```

-   This will start localserver at `localhost:8000`
